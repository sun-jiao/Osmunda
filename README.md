# Osmunda
The README in both Chinese and English is complete.

中文和英文版本的README均是完整的。

# Introduction 简介

An Android library to uncompress Open Street Map data, write into SQLite, for offline geocoding. 

离线地理编码Android库，使用开放街道地图数据。

# OSM data source 开放街道地图数据源
